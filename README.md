📝 Task Manager Application
Ứng dụng quản lý công việc cá nhân được xây dựng bằng Flutter và Firebase. Dự án tập trung vào việc tối ưu hóa trải nghiệm người dùng trong việc sắp xếp và theo dõi tiến độ công việc hằng ngày.

🚀 Live Demo
Bạn có thể trải nghiệm trực tiếp bản Web của ứng dụng tại đây:
👉 https://taskmanager-ng0604.web.app


✨ Tính năng chính
Quản lý Task: Thêm, sửa, xóa và cập nhật trạng thái công việc thời gian thực.

Đồng bộ hóa Cloud: Dữ liệu được lưu trữ và đồng bộ tức thì qua Cloud Firestore.

Xác thực người dùng: Đăng ký/Đăng nhập an toàn với hệ thống của Firebase.

Giao diện Responsive: Hoạt động mượt mà trên cả thiết bị di động và trình duyệt web.

🛠 Công nghệ sử dụng
Frontend: Flutter (Dart)

Backend: Firebase Authentication

Database: Cloud Firestore

Hosting: Firebase Hosting

⚙️ Cài đặt để chạy Local
Nếu bạn muốn chạy dự án này trên máy cá nhân:

Clone repository:
git clone https://github.com/USERNAME/task_managerment.git

Cài đặt dependencies:
flutter pub get

Cấu hình Firebase:
Thay thế file google-services.json (Android) và GoogleService-Info.plist (iOS) bằng cấu hình từ dự án Firebase của bạn.

Chạy ứng dụng:
flutter run
